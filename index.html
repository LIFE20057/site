// Features

Everything you need,
nothing you don't.

💬
AI Chat Interface

Claude.ai-inspired chat as the primary interface. Just describe what you want — the AI handles the HTML.

🤖
Multi-Agent Pipeline

PlannerAgent → CoderAgent → ReviewAgent. Every edit goes through a three-step AI review before it lands.

👁
Live Preview

Instant iframe preview after every AI edit. See your changes in real time, no refresh needed.

🕐
Version History

Zip snapshots before every AI edit. One click to roll back to any previous state.

📁
Multi-Project

Manage multiple websites from a single dashboard. Each project isolated with its own file tree.

🔒
Secure by Default

JWT auth, rate limiting, path traversal prevention, HTTPS support. Production-ready out of the box.

// How it works

Three agents,
one perfect edit.

01 —
PlannerAgent
Analyses your request and builds an action plan
→
02 —
CoderAgent
Generates HTML/CSS/JS file actions
→
03 —
ReviewAgent
Validates and fixes any issues before saving
